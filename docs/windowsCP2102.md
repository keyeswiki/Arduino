# **CP2102驱动安装--Windows**

下载: [https://fs.keyestudio.com/CP2102-WINDOWS](https://fs.keyestudio.com/CP2102-WINDOWS)

进入网址下载 ![image-20230426100746100](./media/image-20230426100746100.png)并解压到 ![image-20230426100852988](./media/image-20230426100852988.png). 

解压得到 ![image-20230426101012182](./media/image-20230426101012182.png)， 请留意该文件夹的路径。

进入“计算机-----属性-----设备管理器”。

![img](./media/wps5.jpg)

页面上的黄色感叹号表示安装不成功，双击![img](./media/wps6.jpg)选择 “**更新驱动...**”。

![img](./media/wps7.jpg)

点击“**浏览我的电脑寻找驱动程序**”找到下载的Arduino软件。

![img](./media/wps8.jpg)

Arduino软件安装包中有一个DRIVERS文件夹，请打开该文件夹查看CP210X系列芯片的驱动程序。

点击“浏览…，然后搜索CP2102的驱动程序，点击“下一步”。

![wps9](./media/wps9.png)

安装成功。

![img](./media/wps10.jpg) 

打开设备管理器，黄色感叹号消失，说明CP2102驱动安装成功。

![img](./media/wps11.jpg)
